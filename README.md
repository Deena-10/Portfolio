## Getting Started

```bash
# clone the repository
git clone https://github.com/<your-username>/<your-repo>.git
cd <your-repo>

# serve locally (example using VS Code Live Server or Python)
python -m http.server 8000
```

Visit `http://localhost:8000` to preview.

## Deploy with GitHub Pages

1. Make sure GitHub Pages is set to deploy from `gh-pages`.
2. Copy `.github/workflows/deploy.yml` from this repo.
3. Push to `main`. GitHub Actions builds and publishes to `gh-pages`.

## Customization

- Update contact form action URL if needed.
- Replace hero CTA links with resume or scheduling links.
- Adjust theme colors in `styles.css` via CSS variables.
